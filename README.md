# 🎮 O QUE JOGAR?

Este projeto foi desenvolvido com a finalidade de ajudar usuários a encontrar um jogo ideal para se divertir, com base nas respostas fornecidas a uma série de perguntas realizadas pelo sistema.

A proposta é simples: responder algumas questões rápidas e receber uma recomendação personalizada de jogo, tornando a escolha mais prática e divertida.

### 🚀 Objetivo

Facilitar a descoberta de novos jogos e auxiliar usuários indecisos a escolherem algo para jogar de acordo com suas preferências.

### 🧠 Como funciona

O sistema realiza perguntas sobre gostos e preferências do usuário, como:

* Você pretende jogar sozinho?
* Você quer uma experiência com história e progresso?
* Que tal um Soulslike?
* Gostaria de jogar jogos competitivos?
* Que tal jogar um FPS/Tático?
* Gostaria de jogar mundo aberto?

📌 Com base nas respostas, o sistema sugere um jogo adequado para proporcionar uma experiência divertida.
